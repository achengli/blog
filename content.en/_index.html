About this website


I did this blog using Hugo static site generator. Hugo is so useful for minimalist people like me because you don't have to acomplish so much in the style and web structure getting deep into css, html and javascript aspects. You can post directly your markdown files and even using LaTeX sections.

If you want to get a look to the source code of this website, you can visit the repository that is open. Be free to copy the code and edit what you want and take inspiration if it could be for it.

The website is located in achengli/blog.git. The link is also available in the left side menu in Github option.
